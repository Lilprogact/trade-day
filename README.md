# ⚡ Trade Day — live site

This repository hosts the **built** Trade Day app (a single self-contained
`index.html`) and deploys it to GitHub Pages on every push to `main`.

**Live app:** https://lilprogact.github.io/trade-day/

A cinematic 3D Earth with the four forex trading sessions (SYD · TYO · LDN ·
NY) in real time: real sun position, session countdowns, the F1 HSN relay jet
flying the session loop with a flight-cam, red-folder USD news, and live ATC
radio with real nearby aircraft (tap the 📻 chip).

The source code lives in the private `Lils` repository (`trade-day/`
folder); this repo only carries the compiled artifact.
